Inline text elements

Inline text elements are used within headings, paragraphs and list items to markup and format words and phrases.

The following inline elements are available via the Squiz matrix wysiwyg toolbar:

<strong>Strong</strong> is used to indicate strong importance visually and aurally.

<em>Emphasis</em> is used to give text emphasis visually and aurally.

<sub>Subscript</sub> for things like H2O H<sub>2</sub>0

<sup>Superscript</sup> for things like Nike™ Nike<sup>™</sup>

<del>Delete</del> for indicating an edit: in this case a deletion denoted by a strike through <del>strike through</del> as opposed to an addition - see <ins> tag below.

<code>Code</code> for inline code examples: var dog = "poodle"; <code>var dog = "poodle";</code>. Often used within <pre> tag when multiline code example needed.

Here are more inline elements that can only be added via code view in Squiz matrix:

<b>Bold</b> and <i>Italics</i> are similiar to strong and emphasis but are visual only.

The abbr element <abbr></abbr>

<abbr>Abbreviation</abbr> element represents an abbreviation or acronym.
The optional title attribute can provide an expansion or description for the abbreviation. If present, title must contain this full description and nothing else:
